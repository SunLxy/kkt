## Quick Start

Create project, Using the template method

```bash
kkt my-app -e chrome-plugin
cd my-app
npm start
```

**development**

Runs the project in development mode.  

```bash
npm run start
```

**production**

Builds the app for production to the build folder.

```bash
npm run build
```
